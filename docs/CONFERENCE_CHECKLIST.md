# FoxBridge Conference Checklist

Operational checklist for event organizers and volunteers running registration and meal validation with FoxBridge.

**Tip:** Print this page or keep it on a tablet at the registration desk. Check items off as you go.

---

## 1. Before the Conference (1–7 days before)

### RegFox & FoxBridge setup

- [ ] Install and open FoxBridge on the registration laptop
- [ ] Complete the **guided setup wizard** (or confirm setup already finished)
- [ ] On **Connect registration**, enter RegFox API key and page ID — attendees should load automatically
- [ ] Confirm attendee count looks reasonable after setup
- [ ] Spot-check a few registrants against RegFox (names, meal choices, registration type)
- [ ] Test search functionality (name, email, organization)
- [ ] On **Mobile scanner** step, confirm “Mobile scanner ready” (or configure mobile service fields)
- [ ] Use **Connect a phone** from the operations home to verify QR, scanner code, and mobile URL

### Badge printing

- [ ] Confirm badge template (name placement, fields, QR code visible)
- [ ] Print several test badges on real label stock
- [ ] Verify printer paper size matches labels (**2.4" × 3.9"** horizontal / Brother DK rolls)
- [ ] Verify QR codes scan correctly with a phone camera
- [ ] Confirm **Print Badge** opens the system print dialog and completes a print
- [ ] Verify printer preferences are remembered after a successful test print

### Meal validation

- [ ] Test meal validation workflow with a few sample attendees
- [ ] Confirm meal plans display correctly (full, half, bring-your-own)
- [ ] Confirm individual meal buttons appear where expected
- [ ] Confirm dietary restrictions display when provided

### Printer connectivity (choose at least one reliable method)

- [ ] **USB printing (recommended backup)** — test print via USB cable
- [ ] **Bluetooth (optional)** — pair printer and test print if using wireless
- [ ] Label which connection method volunteers should use on event day

---

## 2. Hardware Checklist

Pack and label everything before travel.

- [ ] Primary registration laptop (FoxBridge installed)
- [ ] Backup laptop (FoxBridge installed, same settings if possible)
- [ ] Brother QL-820NWB label printer
- [ ] USB cable (recommended — most reliable)
- [ ] Bluetooth paired on primary laptop *(optional)*
- [ ] Extra DK label rolls (plan for peak registration volume)
- [ ] Extension cord
- [ ] Power strip
- [ ] Laptop charger
- [ ] Mouse *(optional — helps volunteers search faster)*

---

## 3. Registration Desk Checklist

Run through this **each morning** before doors open.

- [ ] Open FoxBridge — operations home should show conference name and status
- [ ] Click **Update registrations** if RegFox changed overnight
- [ ] Verify internet connection
- [ ] Print one test badge and confirm layout and QR code
- [ ] Verify printer is online and has labels loaded
- [ ] Verify badge template matches event expectations
- [ ] Open **RegFox Mobile Check-In** app *(if your event uses it for check-in)*
- [ ] Confirm volunteer understands:
  - [ ] How to search for an attendee
  - [ ] How to select an attendee and review the badge preview
  - [ ] How to click **Print Badge**
  - [ ] Who to ask if something looks wrong

---

## 4. Meal Validation Station

Run through this **before the first meal line opens**.

- [ ] Open FoxBridge on the meal station laptop
- [ ] Test RegFox connection
- [ ] Paste or scan a test QR value (or select a test attendee)
- [ ] Verify meal plan and validatable meals display correctly
- [ ] Verify **Validate meal** buttons appear for individual meals
- [ ] Confirm validation works (button confirms the meal)
- [ ] Test duplicate validation prevention (**Already validated** message)
- [ ] Confirm dietary restrictions display when applicable
- [ ] Brief meal-line volunteers: use **Connect a phone** to open the mobile scanner, or desktop **Validate meal**

> **Note:** Camera-based QR scanning in FoxBridge is planned for a future release. For now, paste the QR value or select the attendee from the list.

---

## 5. During Registration

Quick workflow for each attendee:

- [ ] **Search** for the attendee by name (or email / organization)
- [ ] **Verify identity** — confirm name and details with the attendee
- [ ] **Print badge** — click **Print Badge** and wait for the label
- [ ] **Attach badge sticker** — peel and apply carefully
- [ ] **Answer questions** — direct meal, session, or registration questions to the right desk
- [ ] **Walk-ins** — register in RegFox first, then refresh/sync and print in FoxBridge

**Volunteer reminders:**

- [ ] One attendee at a time — avoid rushing past identity check
- [ ] If the badge looks wrong, do not hand it out — re-check the selected attendee
- [ ] Keep the desk uncluttered; spare labels within reach

---

## 6. End of Each Day

- [ ] Verify approximate badge count vs. expected registrations
- [ ] Review meal validation activity (meals validated vs. meals served)
- [ ] Recharge laptops and any scanners/tablets
- [ ] Reload printer labels if the roll is low
- [ ] Note any printer, search, or sync issues for the next morning
- [ ] Backup FoxBridge data *(future — not yet available)*
- [ ] Confirm readiness for the next day (hardware packed, labels stocked, volunteers briefed)

---

## 7. Troubleshooting

| Problem | What to try |
|---------|-------------|
| **Printer not printing** | Check power, labels loaded, USB cable seated; open print dialog and confirm printer selected; try **Print Badge** again |
| **Printer offline** | Wake printer, check USB or Bluetooth; restart printer; switch to USB if Bluetooth fails |
| **Bluetooth issues** | Re-pair Brother QL-820NWB; move laptop closer; **use USB as fallback** |
| **USB fallback** | Connect USB cable → select Brother printer in print dialog → test print |
| **Wrong badge information** | Confirm correct attendee is selected; check RegFox registration; re-sync by restarting FoxBridge |
| **QR won't scan** | Ensure QR is not smudged; increase contrast (reprint if needed); verify phone camera focus; paste attendee id manually at meal station |
| **RegFox connection lost** | Check Wi‑Fi; click **Update registrations**; confirm API key/page ID in Settings → Reopen setup if needed |
| **Mobile scanner not working** | Operations home → **Connect a phone**; confirm scanner code; use **Update registrations** |
| **Need to change printer or credentials** | Settings (gear) → **Reopen setup wizard** |
| **Duplicate attendee names** | Use email or organization in search; confirm identity verbally before printing |
| **Meal validation missing meals** | Confirm attendee’s registration in RegFox includes meal selections; refresh attendee list |

**Escalation:** If issues persist after USB + restart, switch to the backup laptop and notify the event tech lead.
